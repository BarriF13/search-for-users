### Github User search 
- It is a web app which can search github and find users
- It can show :
  - Profile picture
  - Public repos
  - Public Gits
  - Followers
  - Following 
  - Location
  - website or blog
  - Member since
  - 5 latest repos and if they are started, watched or forked
#
## Tech stack 
- Javascript - JS learning objective:
  - Fetch API
  - Promises
  - Async await 
  - Classes
  - Arrow function
  - Event listeners
  - File separation
- HTML
- Bootstrap
- li Theme
